# 🔲 ESP32-016: Colour Sensor using TCS3200 Sensor Module & Servo Motor 🔲

Detect and identify object colors using the **TCS3200 Color Sensor** with an **ESP32 Development Board**. Based on the detected color, the ESP32 controls a servo motor to sort or direct objects, making this project ideal for learning color sensing, PWM signal processing, and automation.

## 🛠️ Components Required

* [ESP32 Development Board](https://robocraze.com/products/nodemcu-32-wifi-bluetooth-esp32-development-board30-pin)
* [TCS3200 Color Sensor Module](https://robocraze.com/products/tcs3200d-tcs230-color-sensor)
* [SG90 Micro Servo Motor](https://robocraze.com/products/sg90-micro-servo-motor)
* [Jumper Wires](https://robocraze.com/products/f2m-jumper-wires-20cm-40pcs)
* [3.7V 800mAh LiPo Rechargeable Battery](https://robocraze.com/products/3-7v-800-mah-lipo-rechargeable-battery)
* [USB to Micro USB Cable](https://robocraze.com/products/usb-to-micro-usb-cable)
